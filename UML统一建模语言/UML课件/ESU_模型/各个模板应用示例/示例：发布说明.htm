发布说明
  1. 目标

    发布说明的目的是为了介绍该课程注册系统发布版的新功能和变更。本文档还说明了该发布版中已知的问题。
  2. 范围

    本文档描述了为 Wylie College 开发的课程注册系统的 1.0 发布版。
  3. 参考

以下是为课程注册系统的用户提供的补充信息：

    • 用户指南
    • 安装指南
    • 系统管理员指南
    • 新发布版的下载站点：http://www/wyliecollege/it/support
  1. 关于本发布版

    课程注册系统的发布版 1.0 将允许 Wylie College 的学生在线进行课程注册。学生们将能够从本校的个人计算机或通过家里的计算机拨号上网来访问课程信息。

    该发布版支持在线维护所有的学生和教授信息。该学院的注册员将可以在整个注册期内监控课程注册过程。通过现有的财务系统的接口可以自动向学生收费。

    如果您在安装软件或使用系统遇到困难，请拨打电话 339-329-8888 或发电子邮件到support@wyliecollege.com 和 Wylie College 技术支持中心联系。

    要获得您的登录 ID，请拨打电话 339-329-8888 与该技术支持中心联系。
  2. 兼容产品

课程注册系统包括在客户个人计算机上运行的客户机程序。个人计算机的最低配置要求为：

    • 486 微处理器或更高
    • 32 MB 内存
    • 20 MB 磁盘空间
    • Windows 95/98/NT
    • Internet 浏览器：Netscape 4.0.4 或者 Internet Explorer 4.0 或更高版本
    • Internet 拨号上网

课程注册系统还包括在 Wylie College 的 Unix 服务器上运行的服务器程序。它与以下系统进行交互：

    • 财务系统（发布版 2.3）
    • 课程目录数据库系统（发布版 2.6）
  1. 升级

    发布版 1.0 的安装说明包括在安装手册中，也可从以下 Web 站点获得：http://www/wyliecollege/it/support。

  2. 新功能

发布版 1.0 中提供的核心功能集有：

  • 在线课程注册
  • 维护所有学生记录
  • 维护教授信息
  • 自动向学生收费
  • 教授登记要讲授的课程
  • 输入学生成绩
  • 访问学生成绩报告单
  • 通过个人计算机远程或本地访问
  1. 已知错误和局限性

    本节列出发布版 1.0 的已知问题并说明有关的解决办法。如果您需要更进一步的帮助，请与我们的技术支持部门联系。

    1. 口令被拒绝

      口令区分大小。

      解决办法：如果系统拒绝您的口令，那么请使用大写和小写字母再输入一遍。
    2. 核查课程的先决条件

      系统将无法正确地核查所需的课程先决条件是否具备。系统显示“不满足课程先决条件”的错误提示消息，并且禁止学生进行课程注册。

      解决办法：如果遇到这类问题，请和 Wylie College 注册员联系，让他们帮助您注册。
    3. 访问课程信息的延时

      在系统使用的高峰期，系统的响应时间可能会受到影响。尤其是访问课程目录中的课程信息会变得很慢。

      解决办法：为避免延时，请在下午 5:00 以后注册。
    4. 非本州的学生

      非本州的学生的地址信息无法正确保存和显示。

      解决办法：将所居住的州或省的信息和国别信息一起输入到地址的第 4 行。
    5. 和 Internet Explorer 不兼容

当在 Internet Explorer 中打开多个图文框时，会遇到一些问题。

解决办法：关闭不使用的图文框。如果仍无法解决问题，请先退出应用程序然后再重新启动。
